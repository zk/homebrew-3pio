# homebrew-3pio

Homebrew tap for [3pio](https://github.com/zk/3pio) - a context-optimized test runner adapter for AI agents.

## Installation

```bash
brew tap zk/3pio
brew install 3pio
```

## Usage

After installation, you can use 3pio:

```bash
3pio npm test
3pio npx jest
3pio npx vitest run
3pio pytest
```

## Updating

```bash
brew update
brew upgrade 3pio
```

## Uninstallation

```bash
brew uninstall 3pio
brew untap zk/3pio
```
